Convert Fixed-Width to CSV

Use this tool to convert Fixed Width into CSV (Comma Separated Values).

Fixed width (or flat file) is column oriented data where each field starts in a certain column and is a fixed length.
For each field, you must enter the position (starting at 1), followed by the field length, followed by the field name.

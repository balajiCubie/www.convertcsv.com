What can this tool do?

Split text files into multiple files and download as a zip file. Load your text or CSV file into the input area, set options, then press Split File and Download.
  • Split CSV File or Split Text File
  • You can choose to split either by setting the maximum number of lines in a file, or by the number of files.
  • If loading a file or URL, it is best to set the options BEFORE loading the file.
  • Because this tool runs in the browser, it works best for files below 2 MB.
